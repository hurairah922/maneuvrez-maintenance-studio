# Done Feature Specs

This directory stores completed implementation specs for Maneuvrez Maintenance Studio.

## Rules

- Move completed specs from `specs/features/active.md` into this directory.
- Use numbered filenames.
- Keep implementation notes, acceptance status, and follow-up issues.
- Do not delete old specs.
- If a completed feature needs more work later, create a new active spec instead of rewriting the completed one.

## Filename Format

Use this format:

    001-plugin-shell.md
    002-admin-settings-foundation.md
    003-maintenance-router.md

## Completion Checklist

Before moving a spec here, confirm:

- The feature works.
- The acceptance criteria passed.
- Security rules were followed.
- Important implementation notes were added.
- Follow-up tasks were documented.
